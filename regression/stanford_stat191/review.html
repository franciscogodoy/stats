Heights of husbands and wives¶

   # Load and attach the data in R
   # The sep indiciates that they are comma separated
   # the header option indicates that the first line
   # contains the variable names

>    url = 'http://stats191.stanford.edu/data/heights.table'
>    heights <- read.table(url, sep=',', header=T)

   # Tell R to put WIFE and HUSBAND in R's toplevel namespace
>    attach(heights)

   # Fit simple linear regression model
>    height.lm = lm(WIFE ~ HUSBAND)

>    plot(HUSBAND, WIFE, pch=23, bg='red', cex=2)

   # Add a line to the plot indicating the best
   # fitting line

>    abline(height.lm, lwd=3, col='yellow')
>    y = 4

It’s also a good idea to get into the habit of removing some variables from the toplevel name space, otherwise you will later get some messages about masking.

> heights2 = read.table(url, header=T, sep=',')
> attach(heights2)

	The following object(s) are masked from heights :

	 HUSBAND WIFE 


# remove them from the namespace

> detach(heights)
> detach(heights2)

Right-to-work example¶

Let’s load the data into R and put the variables into its top namespace.

# This next example is from the textbook
# This time, the values are tab ("\t") separated

> url = "http://www.ilr.cornell.edu/~hadi/RABE4/Data4/P005.txt"
> rtw.table <- read.table(url, header=T, sep='\t')

# This tells R where to find the variables named in the table
> attach(rtw.table)

We can look at boxplots of variables stratified by the right-to-work variable.

# Cost-of-living

> boxplot(COL ~ RTWL, col='orange', pch=23, bg='red')

# Income

> boxplot(Income ~ RTWL, col='orange', pch=23, bg='red')

We can plot continuous variables against each other

# Unionization rate vs. cost-of-living

> plot(URate, COL, pch=23, bg='red')

# Unionization rate vs. income

> plot(URate, Income, pch=23, bg='red')

Finally, we can look at all pairwise plots

> pairs(rtw.table)

New York City, stands out, so let’s remove it. It is the 27th observation, so we use the -27 notation to indicate we want to remove this row from the table.

> pairs(rtw.table[-27,], pch=23, bg='red')

Remember our good habit of detaching...

> detach(rtw.table)

One sample problem¶

This next example goes through some of the computations used in introductory statistics classes. You can read a little more about it on the Data and Story Library (DASL) http://lib.stat.cmu.edu/DASL/Datafiles/Calcium.html.

# Read data into R
# This is an even more advanced use of read.table
# We want to skip the first 28 lines of the file that contain
# some description of the data, and the data is only
# 21 rows long.

> url = 'http://lib.stat.cmu.edu/DASL/Datafiles/Calcium.html'
> calcium.table <- read.table(url,header=T,skip=28,nrow=21)

# Attach the table so R can find the variables Decrease and Treatment

> attach(calcium.table)

# Numerical summaries of the two groups

> treated <- Decrease[(Treatment == 'Calcium')]
> placebo <- Decrease[(Treatment == 'Placebo')]

> print(summary(placebo))
    Min.  1st Qu.   Median     Mean  3rd Qu.     Max. 
-11.0000  -3.0000  -1.0000  -0.2727   2.5000  12.0000 
> print(summary(treated))
   Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
  -5.00   -2.75    4.00    5.00   10.75   18.00 

# Also, each piece of information can be extracted individually

> min(treated)
[1] -5
> max(treated)
[1] 18
> median(treated)
[1] 4
> print(quantile(treated, probs=c(0.25,0.75)))
  25%   75% 
-2.75 10.75 
> sd(treated)
[1] 8.743251
> var(treated)
[1] 76.44444

Boxplots are common graphical summaries of a sample, showing the median and inter-quartile ranges.

> boxplot(Decrease ~ Treatment, col='orange', pch=23, bg='red')

Histograms are also useful summaries of the data.

> hist(treated, main='', xlab='Decrease', col='orange')

> hist(placebo, main='', xlab='Decrease', col='orange')

# Get confidence interval for population
# mean Decrease (ignoring Treatment)

> CI <- c(mean(Decrease)-qt(0.975,20)*sd(Decrease)/sqrt(21),
+ mean(Decrease)+qt(0.975,20)*sd(Decrease)/sqrt(21))
> print(CI)
[1] -1.261000  5.737191

You can do these calculations with the builtin-function called t.test. For instance, to test whether the mean decrease is 0:

> print(t.test(Decrease))

	One Sample t-test

data:  Decrease 
t = 1.3342, df = 20, p-value = 0.1971
alternative hypothesis: true mean is not equal to 0 
95 percent confidence interval:
 -1.261000  5.737191 
sample estimates:
mean of x 
 2.238095

The two sample T-test can either be done with the assumption of equal variances in groups or without. The usual one that is taught in introductory classes has the variances assumed to be equal.

> print(t.test(Decrease ~ Treatment, var.equal=T))

	Two Sample t-test

data:  Decrease by Treatment 
t = 1.6341, df = 19, p-value = 0.1187
alternative hypothesis: true difference in means is not equal to 0 
95 percent confidence interval:
 -1.480770 12.026224 
sample estimates:
mean in group Calcium mean in group Placebo 
            5.0000000            -0.2727273

As discussed in the notes, the two sample problem is really a regression model. We can therefore fit it with the function lm. Note that the coefficient TreatmentPlacebo is the difference between the group Calcium and Placebo. Also, note that the p-values are the same.
